# BannerGrabber

BannerGrabber is a simple port scanner and banner grabber written in Python. 

It is not intended for professional or practical use, but rather as a fun little project for myself, and perhaps, maybe someone uses my code for his or her work. 

Credits to [nmap](https://github.com/nmap/nmap) for insipiration for this tool. 

Make sure to scan on authorized networks only!
